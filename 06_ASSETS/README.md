# Assets - APP EnergyAI

## Estrutura de Assets

```
06_ASSETS/
├── IMAGENS/         → Screenshots, mockups, banners
├── ICONS/           → Ícones do app (favicon, PWA icons)
└── TEMPLATES_PDF/   → Templates visuais do relatório PDF
```

## Ícones Necessários (PWA)

| Arquivo | Tamanho | Uso |
|---------|---------|-----|
| favicon.ico | 32x32 | Aba do browser |
| icon-192.png | 192x192 | PWA Android |
| icon-512.png | 512x512 | PWA Splash |
| apple-touch-icon.png | 180x180 | iOS home screen |
| og-image.png | 1200x630 | Compartilhamento social |
